#### 迭代一实现功能（本次，截至5月24日）

本文档为迭代一的更新文档，实现具体功能如下：

| 功能               | 描述                                                  |
| ------------------ | ----------------------------------------------------- |
| 用户登录、登出     | 用户可以通过【我的】页面实现本小程序的登录与登出      |
| token维持状态      | 程序设置两个小时的token时效，有效期内用户无需再次登录 |
| 用户可发布招聘信息 | 用户可以发布招聘信息                                  |
| 用户可修改招聘信息 | 用户可以查看并且可修改发布的招聘信息                  |
| 附：可删除招聘信息 | 用户可以删除发布的招聘信息                            |
